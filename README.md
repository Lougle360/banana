# Nano Banana 🍌

> 革命性的AI图像编辑工具 - 让修图像聊天一样简单

Nano Banana（Gemini 2.5 Flash Image）是Google开发的革命性AI图像编辑工具，支持一句话编辑、保持人物一致性、多图融合等功能。

## ✨ 核心特性

- **🎯 惊人一致性** - 同一角色多场景不变脸，完美保持人物特征
- **🔄 多图融合** - 支持最多14张图片无缝缝合，自动处理光影和透视
- **💬 口语化编辑** - 像聊天一样给出指令，轻松完成复杂修图
- **🎨 3D公仔化** - 爆火的盲盒手办风格，打造专属数字分身
- **📝 精准文字渲染** - 告别AI绘图乱码，清晰呈现文字内容

## 🚀 快速开始

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/your-username/banana.git
cd banana

# 使用 Node.js serve（推荐）
npx serve -s . -p 8080

# 或使用 Python
python -m http.server 8080
```

然后在浏览器中打开 `http://localhost:8080`

### 直接打开

也可以直接双击 `index.html` 文件在浏览器中打开。

## 💡 使用技巧

### 1. 叠加式对话
不要一次性描述所有要求，采用渐进式修改：
```
第一步：画一个宇航员在火星
第二步：给他的头盔加上夕阳反光
第三步：把宇航服换成深蓝色并加上NASA标志
```

### 2. 角色身份锁定
```
"保持图1中人物长相完全不变，让他换上西装出现在雨后伦敦街头"
```

### 3. 3D公仔化万能公式
```
[人物描述] + Q版/1:7比例 + PVC/树脂材质 + 压克力底座 + 三点布光
```

## 🔧 技术原理

Nano Banana 采用自回归架构而非传统的扩散模型：
- **Token预测** - 将图像视为视觉Token序列进行预测
- **原生多模态** - 文字和图像理解深度融合
- **视觉推理** - 具备物理常识和空间逻辑
- **SynthID水印** - 不可见的数字水印确保AI生成内容溯源

- 严格的内容安全过滤
- 禁止生成公众人物形象（防深伪）
