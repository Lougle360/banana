# Nano Banana 图片生成器使用说明

## 运行步骤

### 1. 安装依赖

首先确保你已安装 Node.js，然后在项目目录运行：

```bash
npm install
```

### 2. 启动服务器

```bash
npm start
```

### 3. 访问网站

打开浏览器访问：http://localhost:8080

## 功能说明

- 输入图片描述文字，AI 将为你生成对应的图片
- 点击示例提示可以快速填充
- 生成过程中会显示加载动画
- 支持下载生成的图片
- 支持重新生成

## 注意事项

- 首次运行需要等待依赖安装完成
- 图片生成可能需要几十秒的时间
- API key 已配置在服务器端，确保安全性
- 如果遇到生成失败，请尝试重新生成或修改提示词

## 开发模式

如果需要热重载功能，可以运行：

```bash
npm run dev
```
